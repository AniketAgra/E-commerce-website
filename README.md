## Features

- User signup / login
- Home page with all featured categories
- Product listing page
- Filters
  - Price range
  - Category
  - Customer rating
  - Availability
- Wishlist management
- Cart Management
- Search Products
- Single Product Page
- User Profile Page
- Address Management 
- Orders page 
- Payment Integration
